\exercise{}{0}
\solution
The $\lambda$-term \textbf{sortincreasing} is applied to a list of Church numerals which returns a list with the same elements in ascending order. The $\lambda$-term is defined the following way (we use the $Y$-combinator):
\begin{align*}
    \text{\textbf{sortincreasing}}\equiv &Y(\lambda bcd.(\lambda fx.f(x x))\\
    &(\lambda fx.f(x x))(\lambda ef.f(\lambda ghi.g(\lambda j.h(\lambda kl.kj(ikl)))\\
    &(hi))e(\lambda gh.h))(\lambda e.d)(\lambda e.b(\lambda f.e(f(\lambda ghi.hg)(\lambda gh.cfh)))))
\end{align*}
Reference: \href{https://codegolf.stackexchange.com/questions/55231/sort-a-list-of-numbers-on-the-\%CE\%BB-calculus}{https://codegolf.stackexchange.com/questions/55231/sort-a-list-of-numbers-on-the-\%CE\%BB-calculus}
